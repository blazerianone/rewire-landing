  • Stateless LLMs & adding memory

  • System prompts for personality
  • API security & .env files
Session 2 — Build a Chatbot
  • Domain scoping — narrow beats broad
  • Wire persistent memory
  • Tone iteration & personality tuning
  • Token pricing & cost awareness
→ Ship: Chatbot with Memory & Personality (tutor, journaling companion) "I can shape how AI behaves."

03 Tools, Agents & AI That Does Things
Why it matters: Chatbots answer questions. Agents take action. This is where AI starts doing real work.
Session 1 — What Is an Agent?
  • Agent vs workflow vs chatbot
  • Function calling — giving AI tools
  • The agent loop: think → act → observe
  • Claude Code & Cursor as agent examples
Session 2 — Build an Agent
  • Pick a real task for your agent
  • Design the toolset
  • Write reliable agent prompts
  • Failure testing & edge cases
→ Ship: Agent That Completes a Real Task (research assistant, data processor) "I built AI that takes actions."
04 Integration — MCP & Automation
Why it matters: AI in a sandbox is a demo. AI connected to your real tools — Gmail, Notion, Slack — is a product.
Session 1 — MCP in Practice
  • What MCP is & why it matters
  • Connect Gmail / Notion / Slack / Drive
  • Safe integrations & permissions
  • Building context-aware AI tools
Session 2 — Automation with n8n
  • Triggers & actions — visual building
  • LLMs inside automation flows
  • Connecting multiple services
  • Reliability & error handling
→ Ship: Automation Bridging Two Services (inbox triage, content engine) "My AI works inside tools I already use."
